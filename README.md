# WereMake
Weremake is designed to be tooling to work with current and future C++ projects to help simplify the process.

## Working on Weremake
`git clone --recurse-submodules https://github.com/wlfstn/weremake`  
`git submodule update --init --recursive` (if you accidentally cloned without recursion)

# Using Weremake

## Files
filetype `.weremake`

## CLI
commands:
- `--setup`: This creates a new `.weremake` file
